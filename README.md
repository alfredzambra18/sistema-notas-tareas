# 📝 Sistema de Notas y Tareas

Mi primer proyecto en Python - Un programa para crear, guardar y gestionar notas personales.

## 🎯 Características

- ✅ **Crear notas** - Escribe notas con título y contenido
- ✅ **Ver todas las notas** - Lista todas tus notas guardadas
- ✅ **Buscar notas** - Encuentra notas por título
- ✅ **Eliminar notas** - Borra notas que ya no necesitas
- ✅ **Guardar datos** - Los datos se guardan automáticamente en un archivo

## 🚀 Cómo usar

### Requisitos
- Python 3.6 o superior

### Instalación
```bash
# Clona el repositorio
git clone https://github.com/alfredzambra18/sistema-notas-tareas.git

# Entra en la carpeta
cd sistema-notas-tareas

# Ejecuta el programa
python notas.py
```

### Menú del programa
```
=== SISTEMA DE NOTAS Y TAREAS ===
1. Crear nueva nota
2. Ver todas las notas
3. Buscar nota
4. Eliminar nota
5. Salir
```
